## Important findings / gotchas
- **Parser is Russian-headings-only.** `detectSection` matches `ОТТАЧИВАЕМ` /
  `ХРИСТИАНСКАЯ`. An English `mwb_E_202605` import was tested and produced no
  `living_christians` section (so no mid_song, and empty parts could pick up
  English titles). **Import the Russian workbook only** until the EN/DE parser
  ships (now a BACKLOG item). A re-import of the RU file heals any English
  titles on empty templates.
- **Re-import is assignment-safe** (verified in `mwb-import.service.ts`): it
  loads existing assignments by partKey, *creates* only when the partKey is
  missing, *updates* only empty templates (no publisher + no assistant), and
  **never deletes**. So backfilling a new partKey (mid_song) via re-import is
  safe; assigned parts are never touched.
- Sisters' field-service capabilities (`fs_starting_conversation`,
  `fs_following_up`, `fs_making_disciples`, `fs_explaining_beliefs`) were
  bulk-enabled for all 60 sisters via SQL (`fs_talk` is brother-only, skipped);
  Lionel prunes per-sister in the UI.
- i18n: 616 keys, consistent across ru/en/de.
